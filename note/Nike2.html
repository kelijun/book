# 耐克售罄下单流程

# 第一步

通过商品链接 (opens new window)查看是否有库存，下图左边是售罄，右边是有货状态。有货按照有货流程 下单，售罄按照该页面流程下单。

# 第二步

⚠️

除360浏览器外，一个浏览器只能两个无痕窗口，你表面六个无痕窗口，实际上只有一个有效账号。 必须通过返利链接下单，否则下了等于白下。

浏览器开无痕窗口，一个浏览器可开两个无痕窗口，推荐谷歌(chrome)、火狐、其它浏览器极速版。输入返利链接 (opens new window)地址，点击购物车，再点击登录，输入账号密码，但是不要点击登录。

# 第三步

等我通知有货，再点击登录结算下单。如果购物车被清空不要管，继续下一个账号，最后再用链接选择性别尺码加购物车。你可以刷新商品链接 (opens new window)查看有库存。

# 第四步

下单成功后，点击返回即可出现订单号，按照账号---密码---订单号---邮箱号格式发送给我，切勿支付，当天晚上结算佣金。

# QA

Q：怎么拿账号？

A：提前一天晚上跟我讲，能下几个账号，我给你几个账号，说一次就可以了，想多下或者少下提前跟我讲。

Q：密码错误怎么办？

A：密码错误就帮忙标记密码错误，找我重新拿账号。

Q：放货时间是什么时候？

A：8点6分50秒放货，放货时间20秒。闲人在8点半之前可以刷新提交订单，中间会随机放货。

Q：没有保存邮箱怎么办？

A：随便找一件商品，点击购买，进入结算页面复制邮箱号，切勿点击下单。

Q：没有抢到怎么办？

A：没有抢到，不扣佣金，不承担任何责任

Q：服务器怎么拿？

A：每天6个以上的订单，提前一天晚上找我索要服务器。

Q：订单号在哪里？

A：订单列表在个人中心。
