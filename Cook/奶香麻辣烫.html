      # 奶香麻辣烫

      在外面吃顿麻辣烫挨了社会毒打，冰柜上写着人均十五的麻辣烫，居然吃了30大洋！！！味道还不咋地。查了一上午资料，麻辣烫竟然要火锅底料，我都有火锅底料了，直接打火锅不好？？？潜心研究一上午，终于搞出来了炒鸡好吃的不用火锅底料的奶香麻辣烫。仅仅需要

      # 厨具

      名称 备注
      煮锅 不建议使用26cm以上的平底锅

      # 用料

      名称 分量 备注
      大蒜 2瓣
      生姜 3片
      大葱 1cm
      辣椒 5个 喜欢吃辣可以加
      香叶 3片
      八角 5个
      花椒 13颗 喜欢吃麻可以加
      生抽 1勺
      料酒 1勺
      蚝油 1勺
      冰糖 2颗 白砂糖一勺
      牛奶 250ml 奶粉也可以，麻辣烫都是奶粉调制的
      食用盐 1小勺 自带的小勺
      芝麻香油 3滴

      # 步骤

      # 步骤一

      准备好食材，洗干净切好装盘备用。

      # 步骤二

      丸子类冷水下锅，煮5分钟。当然也能最后一起煮，个人感觉丸子类煮的时候有奇怪的味道。

      # 步骤二

      葱蒜姜切片

      # 步骤三

      热锅倒油，葱蒜姜（辣椒）炒出香味，加入800ml清水、香叶、八角、（花椒）、生抽一勺、料酒一勺、蚝油一勺、冰糖一勺、盐一小勺、芝麻香油3滴、（食用橄榄油半勺），中火熬15分钟后捞出香料。水不多了，再加入清水200ml、牛奶500ml，保持水牛奶比例在2:1以下，我的是1:1。

      # 步骤四

      放入食材，丸子类没有先煮开先放丸子类煮5分钟，接着放入素菜再煮3分钟。

      # 步骤五

      出锅～撒上芝麻、辣椒拍照发朋友圈
