# 小米运动接口文档

# 注：仅供学习,切勿用于其他用途。

# 1 规范说明

# 1.1 通信协议

HTTPS协议

# 1.2 请求方法

POST请求

# 1.3 字符编码

HTTP通讯及报文BASE64编码均采用UTF-8字符集编码格式。

# 1.4 公共参数

公共参数（Header）是用于标识产品及接口鉴权的参数，每次请求均需要携带这些参数：

参数名称 类型 要求 描述
callid string 默认 1609903974495
x-request-id string 默认 E5C09709-BF56-4587-BB32-4A99B9F11678
appplatform string 默认 ios_phone
user-agent string 可选 MiFit/4.8.1 (iPhone; iOS 14.2.1; Scale/3.00)

# 2 接口定义

# 2.1 获取 Code

  • 接口说明： 获取 Code
  • 接口地址： https://api-user.huami.com/registrations/用户手机号(+8618888888888)/tokens

# 2.1.1 请求参数

参数名称 类型 要求 描述
client_id string 默认 HuaMi
country_code string 可选 CN
password string 必填 密码
r string 可选 AD720B424464796E82D914810DDF1A0D00FD966F01000000D4D9900681AC5A16
redirect_uri string 默认 https://s3-us-west-2.amazonaws.com/hm-registration/successsignin.html
state string 可选 REDIRECTION
t string 可选 当前UNIX时间戳
token string 默认 access
token string 可选 refresh

# 2.1.2 返回结果

参数名称 类型 描述
:status int 响应码
date date 时间
content-length int 长度
location string HuaMi

示例：

:status: 303
date: Wed, 06 Jan 2021 09:10:14 GMT
content-length: 0
location: https://s3-us-west-2.amazonaws.com/hm-registration/successsignin.html?region=cn-northwest-1&access=N-aGT2u1h43342340eUxTs-1h&country_code=CN&expiration=1612221014

注意

code 是 location 参数中 access 的值。参考2.1.2的返回结果，code 为 N-aGT2u1h43342340eUxTs-1h

# 2.2 获取登录用户信息

  • 接口说明： 获取登录用户信息
  • 接口地址： https://account.huami.com/v2/client/login

country_code string 默认 国家代码：CN
device_id string 默认 设备ID：A7D4AF58-2B2E-2B2E-2B2E-0DE5D4ABF92D
device_id_type string 默认 设备ID类型：uuid
device_model string 默认 设备平台：phone
dn string 可选 域名：api-user.huami.com,api-mifit.huami.com,app-analytics.huami.com,account.huami.com,api-watch.huami.com,auth.huami.com
grant_type string 默认 授权类型：access_token
lang string 可选 语言代码：zh_CN
os_version string 可选 os版本号：1.5.0
