# 耐克下单流程

# 第一步

⚠️

除360浏览器外，一个浏览器只能开一个无痕窗口，两个及两个以上，实际上只有一个有效账号。 必须通过返利链接下单，否则下了等于白下。

一个浏览器可开一个窗口加一个无痕窗口，推荐谷歌(chrome)、火狐、其它浏览器极速版。输入返利链接 (opens new window)地址，登录账号，进入购物车页面。

# 第二步

提前进入结算页面，复制邮箱保存好，到8点整在刷新页面，时间太久会生成订单失败。

# 第三步

8点6分50秒开始点击下单按钮，不要等，依次点完下单按钮，再看结果。

# 第四步

下单成功后，点击返回即可出现订单号，按照账号---密码---订单号---邮箱号格式发送给我，切勿支付，当天晚上结算佣金。

# QA

Q：怎么拿账号？

A：提前一天晚上跟我讲，能下几个账号，我给你几个账号，说一次就可以了，想多下或者少下提前跟我讲。

Q：密码错误怎么办？

A：找我重新拿账号。

Q：购物车怎么办？

A：找我重新拿账号。

Q：放货时间是什么时候？

A：8点6分50秒放货，放货时间20秒。闲人在8点半之前可以刷新提交订单，中间会随机放货。

Q：没有保存邮箱怎么办？

A：随便找一件商品，点击购买，进入结算页面复制邮箱号，切勿点击下单。

Q：没有抢到怎么办？

A：没有抢到，不扣佣金，不承担任何责任

Q：服务器怎么拿？

A：每天6个以上的订单，提前一天晚上找我索要服务器。

Q：订单号在哪里？

A：订单列表在个人中心。
